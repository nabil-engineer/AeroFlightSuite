# ✈️ AeroFlight Suite

AeroFlight Suite is a Python application designed to manage and simulate flight operations.

The project allows users to:

- Create a new flight
- Select an aircraft from a database
- Calculate flight time
- Calculate fuel consumption
- Calculate fuel cost
- Save flight reports
- View flight history
- Display flight statistics
- Search aircraft by manufacturer or model

---

## Technologies

- Python 3
- CSV
- Object-Oriented Programming (OOP)
- Modular Programming
